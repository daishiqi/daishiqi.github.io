laravel 的redis扩展安装

2017-02-12 • 标签: redis laravel

php扩展网站:http://pecl.php.net/

1.搜redis,选择stable版 复制链接
2.在/src下面wget:http://pecl.php.net/get/redis-3.1.1.tgz

3.解压:tar zxf 下载好的redis安装包
（-z gz格式，-x解压 -f 以文件形式输出 -v输出详情一般可不写）

4.cd 到解压好的redis-3.1.1文件包

5.发现没有config文件,运行/usr/local/php/bin/phpize(找到最适合php的扩展项 )

6.ls当前目录,发现有configure文件 运行./configure–with-php-config=/usr/local/bin/php-config

7.执行 编译make && make install 运行代码的最后一行显示有redis.so的位置 进入到php.ini里面 修改配置选项(如果不知道php.ini不知道在什么位置:php -i |grep php.ini) vim /etc/php.ini 添加redis.so

8.php -m |grep redis(查找php的扩展模块) 如果返回的是redis 那么redis扩展你就安装成功了
